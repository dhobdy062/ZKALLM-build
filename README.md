# ZKA Static Page

This repository contains a single static HTML page that uses Chart.js via CDN.

## Local Preview

- Serve locally with Python: `py -m http.server 5510`
- Open: `http://localhost:5510/index.html`

## Deploy (GitHub Pages)

1. Create a new GitHub repository (public is fine).
2. Add it as a remote and push the `main` branch.
3. Enable GitHub Pages from the repository Settings → Pages → Deploy from branch → `main` / root.
4. Your site will be available at: `https://<your-username>.github.io/<repo-name>/`.
